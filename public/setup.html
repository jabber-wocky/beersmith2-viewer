How to setup

1. Backing up BeerSmith

If you don't have BeerSmith installed already skip to 2. Installing BeerSmith. To start you'll need to backup your files and uninstall BeerSmith.

  1. Make sure BeerSmith is closed.
  2. Go to your My Documents folder, right click on the beersmith2 folder and select cut.
  3. Right click on your desktop and click paste.
  4. Uninstall BeerSmith

2. Installing BeerSmith

  1. Install Dropbox:
    • https://www.dropbox.com/install
  2. Install Dropbox Folder Sync:
    • http://satyadeepk.in/dropbox-folder-sync/
  3. Install Beersmith2 to your dropbox folder. I use:
    • C:\users\<username<\Dropbox\Program Files\BeerSmith2\
  4. Go to your My Documents folder, right click on the beersmith2 folder, select Dropbox folder sync and click Sync with dropbox

3. Restoring backup

If you skipped step 1 proceed to 4. Setting up BeerSmith Viewer website.

  1. Select your beersmith2 folder on your desktop, right click and select cut.
  2. Go to your My Documents folder, select Edit in the top bar and click paste.
  3. If prompted, click Yes to overwrite any files.

4. Installing BeerSmith on other computers (Optional)


You can only have Beersmith open on one computer at a time.

  1. Uninstall BeerSmith if it is already installed on the new computer.
  2. Install Dropbox:
    • https://www.dropbox.com/install
  3. Install Dropbox Folder Sync:
    • http://satyadeepk.in/dropbox-folder-sync/
  4. Go to your My Documents folder. If there is a folder called BeerSmith2 you must delete it.
  5. Create a new folder called BeerSmith2
  6. Right click on the folder BeerSmith2, select Dropbox folder sync and click Sync with dropbox
  7. A prompt will appear that saying "Destination Already Exists. Merge and Overwrite?". Click Yes.
  8. You can open BeerSmith by going to your \Program Files\BeerSmith2 and opening BeerSmith2.exe
    • Note: you can create a shortcut on your desktop by right clicking BeerSmith2.exe, select Send To, click desktop.

5 Setting up BeerSmith Viewer website (Optional)

  1. Download this zip file containing the website files:
    • beersmith-viewer.zip
  2. Extract the files to your Dropbox folder public folder. Normally located:
    • C:\users\<username<\Dropbox\Public
  3. Open a browser and log into your dropbox account:
    • Dropbox
  4. Browse to your Beersmith folder, find your Recipe.bsmx file and click Share link.
  5. Make sure the visibility is set to Anyone with the link can see it.
  6. Copy the underlined part in the image below. The value will be unique for you. In the example below, I would copy bb8ueaxms1wdlsc
  7. Go to your dropbox public folder (C:\users\<username<\Dropbox\Public). Right click on bsmx-keys.js, select open with notepad.
  8. The top of the file contains a link to your beersmith files. Replace the <key> with the value you copied in step 6 on the recipe line. Save the file. It should look like this at the end:
    • var xml_urls = {
      'recipes' : 'https://dl.dropboxusercontent.com/s/bb8ueaxms1wdlsc/Recipe.bsmx?dl=1',
      'hops': 'https://dl.dropboxusercontent.com/s/qr8rlhrwn6jqtor/Hops.bsmx?dl=1',
      'grains' : 'https://dl.dropboxusercontent.com/s/bxfoei32z4gokk2/Grain.bsmx?dl=1',
      'yeasts' : 'https://dl.dropboxusercontent.com/s/7wr7l32s9rtbnp2/Yeast.bsmx?dl=1'
      }
  9. Repeat steps 4-8 for the Hops.bsmx, Grain.bsmx and Yeat.bsmx files. Save the <key>'s on the hops, grains and yeasts lines at the top of bsmx-loader.js.
  10. You can browse the website going the public folder on the website, selecting the index.html file and clicking Copy public link...

5.1 Setting up BeerSmith Viewer website without a public folder (Optional)

If your dropbox account was created after 2012 you will need to follow some extra steps to get the configuration working correctly since new accounts don't have access to the public folder.

  1. Go to your dropbox public folder on the dropbox website (Dropbox\Public). Click on the index.html page, click shared link and copy the link to the file.
  2. Go to your dropbox public folder on your computer (C:\users\<username<\Dropbox\Public). Right click on bsmx-keys.js, select open with notepad.
  3. In the middle of the file you will see a section that contains the links to the html pages at the root of your web folder. It should look like this:
    var page_urls = {
    'index' : 'index.html',
    'ingredients' : 'ingredients.html',
    'recipe_details' : 'recipe-details.html',
    'recipes' : 'recipes.html',
    'setup' : 'setup.html'
    };
  4. You will need to replace the file name 'index.html' to your dropbox shared link. Make sure there are single quotes around the link. For example,
    'index' : 'https://www.dropbox.com/s/gkhlua7xwisayuv/index.html?dl=0',
  5. Now replace www.dropbox.com with dl.dropboxusercontent.com and delete the ?dl=0 at the end of the url.
    'index' : 'https://dl.dropboxusercontent.com/s/gkhlua7xwisayuv/index.html',
  6. Repeat steps 1-5 for each file (ingredients, recipe-details, recipes, setup).
  7. Go to your dropbox public folder on the dropbox website(Dropbox\Public). Click on the bsmx-keys.js page, click shared link and copy the link to the file.
  8. Go to your dropbox public folder on your computer (C:\users\<username<\Dropbox\Public). Open index.html in notepad and scroll to the bottom of the file. There is a link that looks like this:
    <script src="bsmx-keys.js"></script>
  9. Replace the url bsmx-keys.js with your copied shared link. Replace www.dropbox.com with dl.dropboxusercontent.com and delete the ?dl=0 at the end of the url.
    <script src="https://dl.dropboxusercontent.com/s/bn61bajqj2hrd1h/bsmx-keys.js"></script>
  10. Repeat steps 7-9 to ingredients.html, recipe-details.html, recipes.html, setup.html
  11. You can open your website by getting the shared link for index.html, replace www.dropbox.com with dl.dropboxusercontent.com and delete the ?dl=0 at the end of the url.
    • You might want to create a bookmark for easy reference later.
